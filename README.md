# Weather Dashboard

A responsive Weather Dashboard built using HTML, CSS and JavaScript that displays real-time weather information using a Weather API.

## Features

* Real-time weather data
* Search weather by city name
* Displays temperature, humidity and wind speed
* Shows weather conditions and icons
* Responsive and user-friendly design

## Technologies Used

* HTML
* CSS
* JavaScript
* Weather API

## How to Use

1. Enter a city name.
2. Click the Search button.
3. View the current weather details for the selected city.
